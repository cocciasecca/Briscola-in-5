# Crediti delle carte napoletane

Le 40 immagini delle carte incluse in `src/briscola5/web/static/cards/` provengono dalla categoria **Naples deck** di Wikimedia Commons:

https://commons.wikimedia.org/wiki/Category:Naples_deck

File originali caricati da **Trocche100** e rilasciati dall'autore nel **pubblico dominio**.
Le immagini sono state ridimensionate e compresse esclusivamente per velocizzare il caricamento nella rete locale; il contenuto grafico non è stato modificato.
